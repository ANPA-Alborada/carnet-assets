# Carnet Assets

Repositorio de assets graficos para el carnet de ANPA Alborada.

## Assets incluidos

- `LogoAnpa.png`: logo principal de ANPA Alborada en formato PNG con canal alfa.
- `strip.png`: tira grafica en formato PNG con canal alfa.

## Uso

Los proyectos consumidores deben referenciar los assets desde este repositorio sin modificar el archivo original. Si se necesita una variante, debe documentarse en `IDEAS.md` antes de implementarla.

## Repositorio remoto

- Origin: `git@github.com:ANPA-Alborada/carnet-assets.git`
- Rama publicada: `main`
